int sensehat_initHTS221Sensor ( TSenseHAT *  pTSenseHAT)

Configures HST221 sensor before use. The function sets the following values for the registers of the sensor:

  • AV_CONF = 0x1B;
  • CTRL_REG1 = 0x84;
  • CTRL_REG2 = 0x00;
  • CTRL_REG3 = 0x00.
    Parameters
    [in,out]pTSenseHATpointer to the structure that represents the Sense HAT.
    Returns
    int returns 0 for successful operation, 1 for failure.
