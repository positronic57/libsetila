 Holds the raw output values from the sensor for all 3-axes. The array represents the FIFO of the sensor and every third element has the value for the same axis. Detailed Description

A header file from libsetila library. It defines L3GD20 class for communication with ST L3GD20 3-axis digital output gyroscope. There is no support for sensor interrupts, the user side must pull for sensor readings. The measurement are always done for all of the axes. There is no interface for selecting which axes will be included/excluded in the measurement. When FIFO is enabled, only FIFO mode is supported.
