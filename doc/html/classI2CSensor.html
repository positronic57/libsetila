int I2CSensor::attachSensorToI2CBus ( int  i2cbus)

Associate the sensor with the I2C bus which it physically belongs to.

The file descriptor of the opened I2C bus is provided by I2CBus::getI2CBus(). The bus must be already opened using I2CBus::openI2CBus().

Parameters
[in]i2cbusrepresents the file descriptor of the opened I2C bus provided by I2CBus::getI2CBus().
