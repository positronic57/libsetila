int SPI_Slave_Device::read_write ( void *  transmit_buffer,
void *  receive_buffer,
int  buffer_size 
)
overridevirtual

Implements a synchronous full-duplex communication with the SPI slave. The function sends content of the transmit buffer to the slave, while the slave response is written in the receive buffer. Both buffers have the save size.

Parameters
[in]transmit_buffercontent of this buffer will be sent to the slave device
[out]receive_bufferthis buffer contains the response from the slave device
[in]buffer_sizethe size of the receive and transmit buffers.
Returns
0 in case there was no error while performing IO calls on SPI bus, -1 otherwise
