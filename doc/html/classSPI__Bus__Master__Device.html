int SPI_Bus_Master_Device::configure ( SPI_BUS_MODE  spi_bus_mode,
uint32_t  spi_bus_max_speed_hz,
uint8_t  spi_bits_per_word,
uint16_t  spi_delay 
)

Sets the parameters of the SPI bus. If not later overwritten by SPI slaves init/configure function, these parameters will be used by the SPI slave objects for data exchange on the bus.

Parameters
[in]spi_bus_modeSPI mode (polarity and phase of the bus signal).
[in]spi_bus_max_speed_hzMaximal SPI bus speed in Hz. Value dictated by the SPI slave.
[in]spi_bits_per_wordDevice's word size. Defined based on slave devices requirements.
[in]spi_delayhow long to delay after the last bit transfer before optionally deselecting the device before the next transfer.
Returns
