◆ do_one_shot_measurement()

int HTS221::do_one_shot_measurement ( void  )

Initiate the measurement process for humidity and temperature.

The function starts a single acquisition of temperature and humidity by setting the ONE_SHOT bit of CTRL_REG2 register. After the measuring is done, it reads of the temperature and humidity registers and calculates the temperature and the humidity values using the private class functions: calculateRealtiveHumidity() and calculateTemperature().

Returns
int returns an error code in case there is a failure in communication with the sensor, 0 for successful measurement cycle.
Examples:
HTS221.h.

int HTS221::init_sensor ( uint8_t  AV_CONF_value,
uint8_t  CTRL_REG1_value 
)

Configures the sensor by setting the values of the main config registers.

The function sets the content of the registers: AV_CONF and CTRL_REG1 to the desired values. Before writing the new values to the registers, the function checks if the target is HTS221 sensor, then reads the calibration table using readSensorCalibrationTable() and after that proceeds with register changes.

Parameters
[in]AV_CONF_valuethe new value of the AV_CONF register
[in]CTRL_REG1_valuethe new value of the CTRL_REG1 register
Returns
int return 0 for success, ERROR_INIT_HTS221_SENSOR error code in case of a failure
Examples:
pi_sense_hat.cpp.

◆ init_sensor() [2/2]

int HTS221::init_sensor ( void  )

This function initializes the sensor with the following configuration:

  • block data update bit of CTRL_REG1 set to 1 (inhibit the output register update between the reading of the upper and lower register parts);
  • set the output data rate to 12.5Hz;
  • pressure and temperature internal average to: AVGT = 32 and AVGP = 64.
Returns
int 0 in case of success, _INIT_HTS221_SENSOR error code in case of a failure.
Examples:
HTS221.h.

Reads the content of the calibration table from HTS221 sensor.

The start address of the calibration table is HTS221_CALB_0. The MSB bit of the register address is set to 1 for enabling address auto-increment.

◆ set_one_shot_mode()

int HTS221::set_one_shot_mode ( void  )

Configure the sensor for single acquisition of temperature and humidity (ONE_SHOT mode) with the following settings:

  • pressure and temperature internal average values: AVGT = 256 and AVGP = 512;
  • block data update bit in CTRL_REG1 set to 1;
  • power ON the sensor.
Returns
int returns an error code in case of a failure in communication with the sensor, 0 for success.
