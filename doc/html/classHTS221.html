◆ do_one_shot_measurement()

int HTS221::do_one_shot_measurement ( void  )
private

Initiate the measurement process for humidity and temperature.

The function starts a single acquisition of temperature and humidity by setting the ONE_SHOT bit of CTRL_REG2 register. After that get_sensor_readings() function is called for reading the measurement outputs.

Returns
int returns an error code in case there is a failure in communication with the sensor; ERROR_SENSOR_READING_TIME_OUT when there is no measurement outputs available SENSOR_READING_WATCHDOG_COUNTER number of reading attempts; 0 for successful measurement cycle.
Examples
HTS221.h.

◆ read_calibration_table()

int HTS221::read_calibration_table ( )
private

Reads the content of the calibration table from HTS221 sensor.

The start address of the calibration table is HTS221_CALB_0. The MSB bit of the register address is set to 1 for enabling address auto-increment.

Returns
int returns an error code in case there is a failure in communication with the sensor, 0 for successful read.

The registers and their values for achieving the desired sensor configuration are individual for each ST sensor type. The logic will be implemented in the child class that represents the target ST sensor.

Returns
int 0 in case of successful configuration write, appropriate error code otherwise

Set the resolution for the physical quantity(ies) measured by the sensor. Different ST sensors support different measurement resolution(s). The registers used for that purpose varies from sensor to sensor. The functionality must be implemented in the child class following the correct way for the targeted ST sensor.

Parameters
[in]average_1bit coded number of averaged samples for the first physical quantity
[in]average_2bit coded number of averaged samples for the second physical quantity if any
Returns
int 0 for success, error code otherwise
