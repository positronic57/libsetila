int HTS221::initSensor ( unsigned char  AV_CONF_value,
unsigned char  CTRL_REG1_value,
unsigned char  CTRL_REG2_value,
unsigned char  CTRL_REG3_value 
)

Configures the sensor by setting the values of the main config registers.

The function sets the content of the following registers to the desired values: AV_CONF, CTRL_REG1, CTRL_REG2 and CTR_REG3. Before writing the new values to the registers, the function checks if the target is HTS221 sensor, then reads the calibration table using readSensorCalibrationTable() and after that proceeds with register changes.

Parameters
[in]AV_CONF_valuethe new value of the AV_CONF register
[in]CTRL_REG1_valuethe new value of the CTRL_REG1 register
[in]CTRL_REG2_valuethe new value of the CTRL_REG2 register
[in]CTRL_REG3_valuethe new value of the CTRL_REG3 register
Returns
int return 0 for success, ERROR_INIT_HTS221_SENSOR error code in case of a failure
int HTS221::readSensorCalibrationTable ( )
private

Reads the content of the calibration table from HTS221 sensor.

The start address of the calibration table is HTS221_CALB_0. The MSB bit of the register address is set to 1 for enabling address auto-increment.

Returns
int returns an error code in case there is a failure in communication with the sensor, 0 for successul read.
int HTS221::startHumidityMeasurement ( )

Initiate the measurement process for humidity and temperature.

The function starts the measurement process by setting the value of the less significant bit of HTS221_CTRL_REG2 register. After the measuring is done, it reads of the temperature and humidity registers and calculates the temperature and the humidity values using the private class functions: calculateRealtiveHumidity() and calculateTemperature().

Returns
int returns an error code in case there is a failure in communication with the sensor, 0 for successful measurement cycle.
