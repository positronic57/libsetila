Detailed Description

Base class for accessing the device file of the bus master driver. Provides the file descriptor for the opened device file to the slave devices.

Examples:
adt7410_example.cpp, arduino_mkr_env_shield_rev2.cpp, bmp085_bmp180.cpp, HTS221.h, LPS22HB.h, LPS25H.h, mcp9808_example.cpp, moving_average_window.cpp, pi_sense_hat.cpp, and range_finder.cpp.
