◆ get_sensor_readings()

int ADT7410::get_sensor_readings ( )

Reads the temperature value from the sensor.

The status register value is used to check if the temperature registers contains a valid data. The register is read in a loop till the bit RDY is set to 0 or the maximal number of read attempts are reached.

Returns
0 in case of successful data transfer, ERROR_READ_FAILED in case of issue in data read over I2C, or ERROR_SENSOR_READING_TIME_OUT in case the temperature conversion is not finished in the predefined period of time to avoid blocking.
Examples:
adt7410_example.cpp.
