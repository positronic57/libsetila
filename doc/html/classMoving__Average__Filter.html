Detailed Description

template<class tElement>
class Moving_Average_Filter< tElement >

Simple moving average filter implementation using circular buffer as a moving window.

Can be used on real-time data (readings from a sensor) or applied on a vector of already sampled data.

During the initial filling of the circular buffer the sampling window size is equal to the data-set size. All the values in the moving window are equally weighted.

Template Parameters
tElementrepresents a sensor reading as a filter input

moving_average_window.cpp.

tElement Moving_Average_Filter< tElement >::output ( const tElement &  input)

Calculates on the fly the output of the filter as a mean value for the elements in the window.

During the initial filling of the circular buffer the sampling window size is equal to the data-set size. When there are enough samples for filling the circular buffer, the full window size is used for calculating the average value of the samples in the window.
