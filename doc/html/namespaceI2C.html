Detailed Description

Namespace with functions that implement IO requests for I2C bus. Requests are made by the I2C slave device object using its read/write functions.

Read data from I2C slave device starting from a given internal registry.

Parameters
[in]i2c_mastera file descriptor of the bus master device file
[in]i2c_slave_addressaddress of the slave device on the I2C bus
[in]registrystart reading the data from this internal address in the slave device
[out]bufferdata read from the slave device will be written here
[in]buffer_sizenumber of bites to be read from the device
Returns
0 in case there are no errors on the bus, error code in case of communication failure
